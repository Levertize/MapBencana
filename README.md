# 🗺️ MapBencana — Peta Interaktif Rawan Bencana Indonesia

> Aplikasi pemetaan data interaktif berbasis web yang menampilkan titik-titik rawan bencana di Indonesia secara real-time maupun historis.

## ✨ Fitur

- 🗺️ Peta interaktif dengan Leaflet.js
- 📊 Data gempa real-time dari BMKG
- 🌋 Lokasi gunung berapi aktif Indonesia
- 🎨 Choropleth & heatmap visualization
- 🔍 Filter by jenis bencana & rentang waktu
- 📈 Statistik dan ranking provinsi terdampak
- 🌙 Dark mode

## 🧱 Tech Stack

- **Peta:** Leaflet.js v1.9+
- **Visualisasi:** D3.js v7
- **UI:** Vanilla JS + CSS Custom Properties
- **Build:** Vite
- **Deploy:** GitHub Pages

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build
```
